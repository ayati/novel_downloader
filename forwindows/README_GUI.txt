========================================
小説ePubダウンローダー  かんたんガイド
========================================

■ これは何？
　小説サイトのページの URL を貼り付けてボタンを押すだけで、
　その小説を ePub（電子書籍ファイル）としてダウンロードするアプリです。


■ 使い方（3ステップ）
　1. 読みたい小説のページをブラウザで開き、URL をコピーします。
　   （アドレス欄を右クリック →「コピー」）
　2. このアプリを起動します。コピーした URL が自動で入っていることが
　   あります。入っていなければ、入力欄に貼り付けてください。
　3. 大きな「ダウンロード」ボタンを押します。
　   　→ 終わると自動でフォルダが開き、できた ePub ファイルが選ばれた
　   　　状態で表示されます。


■ ファイルはどこに保存される？
　「ダウンロード」フォルダの中の「小説」フォルダに保存されます。
　（保存先はアプリの「詳細設定」で変更できます）


■「WindowsによってPCが保護されました」と表示されたら
　初めて起動するとき、青い画面で警告が出ることがあります。
　これは「作った人が有料の証明書を持っていない」ために出るもので、
　ファイル自体に問題はありません。次の手順で起動できます。

　　1.「詳細情報」という小さな文字をクリックします。
　　2. 下に出てくる「実行」ボタンを押します。

　毎回出るのが気になる場合は、ダウンロードしたファイルを右クリック →
　「プロパティ」→ 一番下の「許可する（ブロックの解除）」にチェック →
　「OK」を押すと、次から警告が出なくなります。


■ うまくいかないとき
　・「このサイトには対応していません」と出た
　　　→ URL が正しいか確認してください。対応サイトはアプリの
　　　　「対応サイトを見る」で一覧できます。
　・「ハーメルンには対応していません」と出た
　　　→ 申し訳ありませんが、ハーメルンはこのアプリでは扱えません。
　・「うまくいきませんでした」と出た
　　　→ インターネットの接続を確認して、もう一度お試しください。
　　　　時間をおくと直ることもあります。


■ 詳細設定（必要な人だけ）
　「詳細設定」を開くと、次のことが変えられます。
　・保存先フォルダ
　・表紙（自動でつくる／サイトの表紙を使う／自分の画像を使う）
　・そのほか（横書き、Kobo端末向け、など）
　普段はそのままで大丈夫です。


■ パソコンに詳しい方へ
　本アプリは付属の novel_downloader.exe（コマンドライン版）を呼び出して
　動作します。コマンドラインから直接使うこともできます。
